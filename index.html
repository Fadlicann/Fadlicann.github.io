Tentang Saya

Hai, Saya Fadli seorang programmer yang memiliki ketertarikan mendalam dalam dunia software engineering, khususnya dalam pengembangan aplikasi web. Saya senang memecahkan masalah dan menciptakan solusi teknis yang efisien serta kreatif melalui pemanfaatan teknologi modern.

Saat ini saya bekerja sebagai Programmer / IT Specialist di Biro Administrasi Pimpinan Sekretariat Daerah Provinsi Sumatera Utara. Dalam peran ini, saya bertanggung jawab untuk merancang, mengembangkan, serta melakukan pemeliharaan berbagai website dan sistem internal yang digunakan untuk mendukung operasional di lingkungan biro.

Sebagai satu-satunya tenaga IT di unit tersebut, saya juga menangani berbagai kebutuhan teknis teknologi informasi, mulai dari pengembangan fitur baru hingga memastikan sistem berjalan stabil dan optimal.

Sebelumnya, saya memperoleh pengalaman melalui studi independen di PT Kinema Synstrans Multimedia dalam pengembangan aplikasi berbasis web dan penerapan teknologi modern pada proyek nyata.

Saya juga pernah magang di PT Kodinglab Integrasi Indonesia, di mana saya bekerja bersama tim pengembang untuk membangun serta memelihara aplikasi web.

Sebagai web developer, saya menguasai teknologi seperti HTML, CSS, JavaScript, PHP, Yii2, dan SQL. Saya selalu memastikan aplikasi yang saya buat tidak hanya fungsional tetapi juga user friendly.

Saat ini saya terus mengembangkan keterampilan untuk menghadapi tantangan baru di dunia software engineering.
