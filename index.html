Tentang Saya

Hai, Saya Fadli seorang mahasiswa tingkat akhir yang memiliki ketertarikan mendalam dalam dunia software engineering, khususnya dalam pengembangan aplikasi web. Saya senang memecahkan masalah dan menciptakan solusi teknis yang efisien serta kreatif. Saya memiliki pengalaman yang berharga melalui studi independen di PT Kinema Synstrans Multimedia, di mana saya belajar banyak tentang pengembangan aplikasi berbasis web dan penerapan teknologi modern dalam proyek-proyek riil. Selain itu, saya juga pernah magang di PT Kodinglab Integrasi Indonesia, di mana saya bekerja dengan tim pengembangan untuk membuat dan memelihara aplikasi web, serta memecahkan berbagai tantangan teknis.

Sebagai seorang web developer, saya menguasai berbagai teknologi seperti HTML, CSS, JavaScript, PHP, Yii2, dan SQL. Saya selalu berusaha untuk mengerjakan setiap proyek dengan pendekatan yang kreatif, memastikan bahwa hasil akhirnya tidak hanya fungsional, tetapi juga menarik dan mudah digunakan. Saat ini, saya sedang fokus untuk mengembangkan keterampilan saya lebih lanjut dan siap untuk menghadapi tantangan baru dalam dunia software engineering.
